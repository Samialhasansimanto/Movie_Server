# 🎬 CineMind - AI Movie Recommendation System

CineMind is an AI-powered movie recommendation website that suggests movies based on mood, genre, and user preferences.

## 🚀 Features

- Mood-based movie recommendations
- Movie search functionality
- Trending movies section
- Hollywood, Bollywood & Korean movies
- Responsive modern UI
- TMDB API integration

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Flask
- TMDB API

## 📂 Installation

```bash
git clone https://github.com/Samialhasansimanto/Movie_Server.git
cd Movie_Server
pip install -r requirements.txt
python app.py
```
